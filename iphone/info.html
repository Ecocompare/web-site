Présentation

ecocompare.com est un portail web et iPhone des eco-produits. Il évalue les produits durant les différentes étapes de leur cycle de vie : fabrication, utilisation ou fin de vie. De chaque étape résulte une note de 0 (aucun intérêt écologique) à 5 (excellent), une note globale affiche la moyenne.

L'application iPhone

Cette application vous permet de visualiser les derniers produits publiés, de scanner un ou plusieurs code barre de produits, de faire afficher la fiche environnementale ou encore de les comparer entre-eux (comparateur).

Mode eco-acteur

Pour les produits non encore référencés, vous pouvez également participer à notre démarche dans la mesure ou les produits les plus scannés feront l'objet d'une demande auprès des fabricants. Vous pourrez retrouver sur le site web, le top 10 des codes barres les plus demandés ainsi que le top 10 des eco-acteurs les plus actifs. La démarche est libre, chaque utiliseur peut décider dans l'onglet "mon compte" de participer ou non à la démarche.
