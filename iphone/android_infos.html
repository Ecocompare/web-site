Présentation

ecocompare.com est un portail web, iPhone et Android des eco-produits. Il évalue les produits durant les différentes étapes de leur cycle de vie : fabrication, utilisation ou fin de vie. De chaque étape résulte une note de 0 (aucun intérêt écologique) à 5 (excellent), une note globale affiche la moyenne.

L'application Android

Cette application vous permet de visualiser les derniers produits publiés, de scanner un ou plusieurs code barre de produits, de faire afficher la fiche environnementale ou encore de les comparer entre-eux (comparateur).

Mode eco-acteur

Pour les produits non encore référencés, vous pouvez également participer à notre démarche dans la mesure où les produits les plus scannés feront l'objet d'une demande auprès des fabricants. Vous pourrez retrouver sur le site web le top 15 des codes barres les plus demandés ainsi que le top 15 des eco-acteurs les plus actifs. La démarche est libre, chaque utiliseur peut décider dans la vue "Compte" de participer ou non à la démarche.


Utilisation


Menu
Affiche les derniers produits publiés sur ecocompare. Vous avez la possibilité de les rajouter dans le comparateur.
Permet de scanner, ou de saisir manuellement, le code barre d'un produit.
Regroupe les produits que vous souhaitez comparer, au nombre de 3 produits maximum.
Permet de rechercher un produit par mot-clés, marque...
Boutons
Visible depuis les onglets Accueil et Scanner, ce bouton permet d'accéder à la page compte pour paramétrer vos coordonnées, choisir une photo et indiquer si vous souhaitez participer à la démarche eco-acteur.
Accessible via les fiches produits, ce bouton permet d'ajouter le produit dans le comparateur.

Confidentialité
